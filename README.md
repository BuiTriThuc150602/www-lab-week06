# www-lab-week06
# Mô tả sơ lược
### - Một web blog đơn giản, cho phép người dùng đăng nhập, xem các bài đăng, comment để tương tác với bài đăng. Ngoài ra còn có thể đăng bài viết của mình cho mọi người thấy.

# Chạy chương trình : 
### - Build dự án với gradle, phiên bản hiện dùng để phát triển là 8.2
### - Spring Boot : Spring Framework 6.x
### - Server Tomcat 10.1.13
